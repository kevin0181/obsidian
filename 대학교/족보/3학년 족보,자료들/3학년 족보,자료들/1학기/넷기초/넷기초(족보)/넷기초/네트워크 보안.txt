  불법 행위로부터 정보를 보호하는것
- 기밀성/무결성

필요성

- 애플: 보안 탄탄함

네트워크 보안의 피해 사례
- 디도스 -> 인터넷 대역폭 추가 확보, 주요 컨텐츠나 데이터 베이스 분산, 인터넷 트래픽 실시간 모니터링 및 즉각 대응
- 월패드 해킹 -> 세대간 망 분리, 매터 도입

개인정보 보호의 필요성
- 개인: 권리와 자유 보호
- 기업: 기업 경영 타격 방지
- 국가: 정부 신뢰성 하락 방지

암호화 기술
대칭 키 암호화: 하나의 키로 모두 작동. 빠른 속도 but 키 공유와 같은 문제
비대칭키 암호화: 서로 다른 키 사용. 대칭키 단점 보완 + 높은 보안성 유지

접근제어 기술
사용자가 프로세스 인증 후 사용.

etc
- 보안 인증 기술: 사용자 본인 인증
- 익명화 기술: 개인정보 식별 불가능하게 가공
- 보안 로그 기술: 시스템 안에서 보안 이벤트
- 가상화 기술: 정보 보호를 위해 가상화 기술 사용
- 위변조 탐지 기술: 데이터의 변경과 조작을 탐지하고 대응

보안 위협 요소의 전망

보안 기술의 전망
